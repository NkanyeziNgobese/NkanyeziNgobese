## About Me

I am a Computer Science Honours graduate and Graduate Test Engineer based in South Africa. I build software systems while also developing strong quality-engineering discipline through app testing, API testing, regression testing, Java automation exposure, and test-environment work.

My profile is not only testing-focused. I am a developer who enjoys designing and building practical tools: adaptive learning platforms, recall-based study systems, finance-technology assistants, simulations, AI/NLP experiments, and data-driven applications.

Testing strengthens how I build because it keeps me focused on reliability, edge cases, user flows, repeatability, maintainability, and whether a system behaves correctly under real use.

## What I Build

* **Application software** - tools, dashboards, learning platforms, and personal productivity systems.
* **AI and learning systems** - adaptive learning, recall-based practice, NLP, and knowledge tooling.
* **Finance-technology tools** - investment-literacy systems, financial planning tools, and decision-support ideas.
* **Simulation and data projects** - systems modelling, congestion analysis, data wrangling, and operational insights.
* **Quality-focused software** - tested, maintainable, and user-aware systems informed by API, regression, and automation testing.
